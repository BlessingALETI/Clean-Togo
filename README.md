Clean Togo

Plateforme intelligente de gestion des déchets ménagers

Description
Clean Togo est une plateforme numérique conçue pour améliorer la gestion du ramassage des déchets ménagers au Togo.
Elle permet de connecter les ménages, les chauffeurs et les administrateurs afin d’optimiser les collectes, les paiements et le suivi des opérations.


Objectifs
améliorer l’organisation des collectes
faciliter la communication entre les acteurs
digitaliser les abonnements et paiements
fournir des statistiques aux administrateurs


Technologies utilisées
#Backend
Java
Spring Boot
API REST

#Frontend Web
Angular

#Application Mobile
Flutter

#Base de données
PostgreSQL


Installation
1. Cloner le projet
git clone https://github.com/ton-repo/clean-togo.git
cd clean-togo

2. Base de données
Créer une base PostgreSQL
Exécuter le script SQL fourni

3. Backend (Spring Boot)
Ouvrir le projet dans IntelliJ IDEA
Configurer application.properties
Lancer l’application

4. Frontend (Angular)
npm install
ng serve

5. Mobile (Flutter)
flutter pub get
flutter run


Sécurité
Authentification avec JWT
Gestion des rôles (ADMIN, CLIENT, CHAUFFEUR)
Protection des données sensibles


Fonctionnalités principales
inscription et connexion des utilisateurs
gestion des abonnements
paiement via Mobile Money
planification et suivi des collectes
notifications
tableau de bord administratif


Structure du projet
/backend : API Spring Boot
/frontend : application Angular
/mobile : application Flutter
/database : scripts SQL
